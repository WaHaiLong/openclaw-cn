| 文件 | 说明 |
|------|------|

## 🚀 GitHub Pages 发布步骤

1. 在 GitHub 创建新仓库（例如 `openclaw-cn`）
2. 上传本目录所有文件
3. 进入仓库 **Settings → Pages**
4. Source 选择 `Deploy from a branch`，Branch 选择 `main`，目录选择 `/ (root)`
5. 点击 Save，等待约 1 分钟
6. 访问 `https://你的用户名.github.io/openclaw-cn/`

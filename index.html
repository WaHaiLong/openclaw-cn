快速开始

5 分钟上手

支持 macOS、Windows 和 Linux。一键脚本自动安装 Node.js 及所有依赖。

01

安装 OpenClaw

使用一键安装脚本（推荐），自动处理 Node 检测和安装：

curl -fsSL https://openclaw.ai/install.sh | bash
02

运行引导向导

配置认证、Gateway 设置和频道连接：

openclaw onboard --install-daemon
03

检查 Gateway 状态

确认后台服务是否正常运行：

openclaw gateway status
04

打开控制台

在浏览器中直接开始聊天，无需额外频道配置：

openclaw dashboard

🍎 macOS 伴侣应用（Beta）

菜单栏快速访问你的 AI 助手。与 CLI 完美配合使用。需要 macOS 15+，通用二进制。
